|---|-----------|
| RF-01 | Monitoreo del estado y versión del sistema vía endpoint `/health` |
| RF-02 | Consulta del catálogo de productos con filtrado por categoría |
| RF-03 | Registro de nuevos productos con validación de datos |
| RF-04 | Actualización de datos de productos existentes |

| RF-08 | Gestión de créditos a clientes y aplicación en cobros |
| RF-09 | Consulta de estadísticas y métricas del inventario |
| RF-10 | Manejo global de errores con respuestas JSON consistentes |
| RF-11 | Apertura y cierre de sesiones de caja con montos inicial y final |

| RF-13 | Aplicación de créditos del cliente en ventas POS |
| RF-14 | Descuento automático de stock al registrar una venta POS |
| RF-15 | Generación y consulta de ticket/recibo por venta |
| RF-16 | Consulta del historial de ventas por sesión de caja |
