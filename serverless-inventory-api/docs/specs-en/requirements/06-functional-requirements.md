# Functional Requirements (FR)

| # | Capability |
|---|-----------|
| FR-01 | System health and version monitoring via `/health` endpoint |
| FR-02 | Product catalog query with category filtering |
| FR-03 | New product registration with data validation |
| FR-04 | Existing product data update |
| FR-05 | Logical product deletion from catalog |
| FR-06 | Full customer management (CRUD) with unique email validation |
| FR-07 | Charge registration with total calculation and automatic stock deduction |
| FR-08 | Customer credit management and application to charges |
| FR-09 | Inventory statistics and metrics query |
| FR-10 | Global error handling with consistent JSON responses |
| FR-11 | Cash session opening and closing with initial and final amounts |
| FR-12 | POS sale registration with product selection and payment methods |
| FR-13 | Customer credit application in POS sales |
| FR-14 | Automatic stock deduction when registering a POS sale |
| FR-15 | Ticket/receipt generation and query per sale |
| FR-16 | Sales history query by cash session |
